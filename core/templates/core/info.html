Система опросов

Функционал:

1) Создание опроса
Пользователь может задать тему опроса и составить несколько вариантов ответа.
2) Участие в опросах и просмотр их результатов
Пользователь может участвовать в опросах, а также просматрировать их результат.
В конце опроса пользователю будет продемонстрирован результат.

Возможные корректировки/улучшения:

1) Добавление тестов
Пользователь будет выбирать кол-во вопросов и для каждого вопроса выбирать кол-во ответов.
2) Cначала кол-во ответов/вопросов при создании опроса/теста будет фиксировано,
но в будущем попробую запрашивать кол-во вариантов ответов/вопросов у пользователя.
